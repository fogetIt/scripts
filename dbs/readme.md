##### DBS(Database System)
- 数据库系统，由数据库及其管理软件组成
    1. 数据库(database)，按一定的数学模型组织、描述和存储数据
        - 较小的冗余
        - 较高的数据独立性和易扩展性
        - 可为各种用户共享
    2. 硬件——构成计算机系统的各种物理设备，包括存储所需的外部设备
    3. 软件——操作系统、各种宿主语言、数据库管理系统及应用程序
    4. 人员
        - 系统分析员和数据库设计人员
        - 应用程序员
        - 最终用户
        - 数据库管理员(data base administrator，DBA)
            - 负责数据库的总体信息控制
